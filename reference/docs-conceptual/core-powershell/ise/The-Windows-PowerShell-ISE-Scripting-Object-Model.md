# <a name="the-windows-powershell-ise-scripting-object-model"></a>Skriptovací objektový model prostředí Windows PowerShell ISE
  Windows PowerShell® Integrované skriptovací prostředí (ISE) zveřejňuje jeho funkce prostřednictvím základní skriptování objektový model, který umožňuje uživatelům k psaní skriptů prostředí Windows PowerShell, které pracují s různé visual a funkční aspekty Windows PowerShell ISE. Skriptovací objekty, jejich vlastnosti a jejich metody jsou definovány speciálně pro tento účel.
